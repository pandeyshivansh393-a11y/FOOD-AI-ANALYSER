# FOOD-AI-ANALYSER
Overview

Food AI Analysis is a web-based application that uses the Gemini API to analyze food items from user input and provide AI-generated insights. The application can identify food items and generate useful information such as nutritional details, dietary suggestions, or other food-related analysis based on the provided input.

Features

- AI-powered food analysis using the Gemini API.
- Simple and user-friendly interface.
- Fast responses with detailed food insights.
- Built using HTML, CSS, and JavaScript.

Technologies Used

- HTML
- CSS
- JavaScript
- Google Gemini API

Getting Started

Prerequisites

- A modern web browser.
- A free Gemini API key from Google AI Studio.

Setup

1. Clone this repository.
2. Obtain a Gemini API key from Google AI Studio.
3. Open the project files and locate the API key section.
4. Replace the placeholder value:

const API_KEY = "YOUR_GEMINI_API_KEY";

with your own Gemini API key.

5. Save the file and open "index.html" in your browser.

Important Note

For security reasons, the actual Gemini API key is not included in this repository. Each user must use their own API key to run the application. This follows standard security practices and prevents unauthorized use of API credentials.
